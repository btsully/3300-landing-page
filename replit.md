# 3300-landing-page

## Project Overview
A simple static HTML landing page.

## Structure
- `index.html` - Main HTML file (the only page)

## Running the App
The app is served via Python's built-in HTTP server on port 5000.

- **Workflow**: "Start application" — `python3 -m http.server 5000`
- **Port**: 5000

## Deployment
- **Type**: Static site
- **Public directory**: `.` (project root)
